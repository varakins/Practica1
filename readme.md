*ООО <<ОБУВЬ>>
Это система которая поозваляет просматревать редактировать удалять создовать товары и заказы.
**Гость
Гость имеет возможность просмптривать товары без фильтрации сортировки и поиска
**Авторизованный клиент
Автризованный клиент может просмптривать товары без фильтрации сортировки и поиска и авторизоватся
**Менеджер может 
-Авторизоватся
-Просмотр товаров с возможностью фильтрации сортировкой и поиском
-Просмотр заказов
**Администратор
-Авторизоватся
-Управление товарами
--Просмтор с фильтрацией сортировкой и пойиском
--Добавление товара
--Удаление товара
--Редактирование товара
-Управление заказами
--Просмотр заказа
--Добавление заказа
--Удаление заказа
--Редактирование заказа
**Запуск проект
Запускаем из папки приложения консол
(powershell)
прописываем
dotnet build
dotnet run
